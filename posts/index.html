第二篇博客

如何利用Hugo创建个人博客？ 什么是Hugo？ Hugo是一种通过Go语言实现的博客生成器，是目前世界上最快的博客生成器。 作为程序员，不应囿于一种程序语言。 如何安装Hugo？ Mac系统直接运行以下代码： brew install hugo hugo version Windows系统先去Hugo release page下载exe文件安装，然后运行 hugo version 如果看到版本号，即代表安装成功 如何使用Hugo搭建博客？ 进入Hugo官网，点击Quick Start，根据官网操作的七个步骤，抄代码搭建 第七步以后其实没有结束，没有交代如何把博客挂到网上。需要让public文件夹自称仓库，然后进行部署，把public上传一个单独的仓库，注意要把public忽略 部署的文件夹，必须是 github的用户名.github.io 上传完毕后，刷新github文件夹页面，点击settings的github pages，找到source下面的按钮，选择master（如果没有选项就不用管）然后保存 得到地址以后，点击网址，个人博客就开通了 如果想换皮肤，官网上可以下载
